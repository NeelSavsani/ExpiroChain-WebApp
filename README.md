## 📌 About The Project

EXPIROCHAIN is an innovative platform designed to manage the complete lifecycle of medicines — from manufacturing to usage, expiry tracking, waste management, and compliance monitoring.

The platform helps pharmacies, hospitals, suppliers, NGOs, and regulatory authorities reduce medicine wastage, improve medicine utilization, and ensure safe disposal practices.

---

## 🚀 Features

- 📦 Medicine Inventory Management
- ⏰ Expiry Date Tracking & Alerts
- ♻️ Medicine Waste Rescue System
- 🏥 Hospital & Pharmacy Management
- 🔐 Secure Authentication System
- 📊 Dashboard & Analytics
- 📄 Compliance Monitoring
- 🌐 User-Friendly Web Platform
